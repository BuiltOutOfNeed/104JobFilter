隱私權政策

最後更新：2026 年 5 月 12 日

104 求職過濾器（以下簡稱「本擴充功能」）是一款 Chrome 瀏覽器擴充功能，協助您在 104.com.tw 上過濾及標記工作列表。本政策說明本擴充功能存取哪些資料，以及如何處理這些資料。

存取的資料

  • 您輸入的關鍵字 — 您在擴充功能彈出視窗中輸入的封鎖與醒目關鍵字。
  • 求職投遞紀錄 — 當您使用「抓取已投遞公司」功能時，本擴充功能會透過您現有的瀏覽器工作階段（Cookie）從 104.com.tw 讀取您的投遞公司名稱。此資料僅在您的瀏覽器本機處理，不會離開您的裝置。
  • 聯絡紀錄 — 當您使用「抓取聯絡我的公司」功能時，本擴充功能會透過您現有的瀏覽器工作階段（Cookie）從 104.com.tw 讀取曾聯絡您的公司名稱。此資料僅在您的瀏覽器本機處理，不會離開您的裝置。

資料儲存方式

所有關鍵字資料均使用 chrome.storage.local 儲存於您裝置的本機。本擴充功能不會將任何資料傳送至任何伺服器。

資料分享

本擴充功能不會收集、傳輸、出售或與任何第三方分享您的個人資料。本擴充功能沒有後端伺服器。

權限說明

  • storage — 用於儲存您的關鍵字清單。
  • tabs — 用於在觸發抓取功能時，向目前的 104.com.tw 分頁傳送訊息。
  • host_permissions (104.com.tw) — 用於執行過濾工作卡片的內容腳本，以及使用您現有的工作階段抓取投遞紀錄。

聯絡方式

如有任何疑問，請至 GitHub 專案頁面提交 Issue。

Privacy Policy

Last updated: 2026-05-12

104 Job Filter ("the extension") is a Chrome browser extension that helps you filter and highlight job listings on 104.com.tw. This policy explains what data the extension accesses and how it is handled.

Data We Access

  • Keywords you enter — block and highlight keywords you type into the extension popup.
  • Job application history — when you use the scraper feature, the extension reads your applied company names from 104.com.tw using your existing browser session (cookies). This data is fetched directly in your browser and never leaves your device.
  • Contacted company history — when you use the contacted companies scraper, the extension reads company names that have messaged you from 104.com.tw using your existing browser session (cookies). This data is fetched directly in your browser and never leaves your device.

How Data Is Stored

All keyword data is stored locally on your device using chrome.storage.local. No data is sent to any server operated by this extension.

Data Sharing

We do not collect, transmit, sell, or share any personal data with third parties. The extension has no backend server.

Permissions

  • storage — to save your keyword lists.
  • tabs — to send messages to the active 104.com.tw tab when the scraper is triggered.
  • host_permissions (104.com.tw) — to run the content script that filters job cards and to fetch your application history using your existing session.

Contact

If you have any questions about this policy, please open an issue on the GitHub repository.
